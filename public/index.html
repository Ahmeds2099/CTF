What Is This Event?

A Fun Cybersecurity Puzzle Challenge

Think of it like an escape room — but digital. You'll receive a USB drive full of mysterious files (images, audio, text files, network logs). Your job is to figure out the hidden clues inside each file using free tools we'll teach you about.

Each clue you find is called a "flag" — submit it on our online portal to earn points. The more flags you find, the higher your rank. It's competitive, educational, and a lot of fun.

  • No coding or hacking experience needed
  • Open to ALL college students from any institution
  • Solo participation — just you and the evidence
  • Challenges range from easy to advanced
  • Learn real cybersecurity skills while competing

Open the files on the Evidence Drive. Use the tools to analyze images, audio, network captures, and encrypted files. Each piece of evidence contains a hidden "flag" string.

Your Forensic Toolkit

Download and practice with these tools BEFORE the event. They're all free and easy to use.

CyberChef Web Tool

The Cyber Swiss Army Knife — paste text and decode Base64, ROT13, hex, binary, and hundreds of other formats instantly. Drag-and-drop easy.

Launch

Wireshark Download

Network traffic analyzer — open .pcap files and read packet data. Look for hidden messages in network communications.

Download

Aperi'Solve Web Tool

Upload any image and it automatically checks for hidden data using multiple steganography techniques. Just drag and drop.

Launch

HexEd.it Web Tool

Hex editor in your browser — open any file and see its raw bytes. Useful for finding embedded text or fixing corrupted file headers.

Launch

Audacity Download

Audio editor — open audio files and switch to spectrogram view. Hidden messages can be encoded as visual patterns in sound frequencies.

Download

ExifTool Download

Reads hidden metadata from files — GPS coordinates, comments, timestamps, and author info that may contain clues.

Download

StegSolve Java App

Cycle through color channels and bit planes of images to reveal hidden visuals. Great for image-based steganography challenges.

Get Tool

Binwalk CLI Tool

Extracts files hidden inside other files — like a ZIP embedded within an image. Run from terminal on Linux/WSL.

Get Tool

Pro tip: Web tools need no installation. For download tools, install them now and try opening a random image/file to make sure they work. You don't want to troubleshoot installations during the event!
